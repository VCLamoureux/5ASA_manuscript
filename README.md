# Pan-repository analysis reveals a drug-activating function of microbial bile acid conjugation
We showed that gut bacterial conjugation of 5-aminosalicylic acid (5-ASA) to bile acids results in activation of the drug, yielding conjugates with greater potency than 5-ASA itself.

# System requirements
RStudio can be installed using the following link (https://posit.co/download/rstudio-desktop/) in less than 15 min on a "normal" computer
Instructions: Click on "download and install R, followed by choose macOS version as the second step. 

Custom scripts were executed on an Apple MacBook Pro running macOS Tahoe version 26.2, equipped with an Apple M2 Max chip (12-core CPU, 38-core GPU) and 64 GB of unified memory.
Package version numbers are listed in the custom scripts itself or avaiable using ?package_name in RStudio (e.g., ?tidyverse)
