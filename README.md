# Pan-repository analysis reveals a drug-activating function of microbial bile acid conjugation
We show that gut bacterial conjugation of 5-aminosalicylic acid (5-ASA) to bile acids results in metabolic activation of the drug, yielding conjugates with greater potency than 5-ASA itself.
